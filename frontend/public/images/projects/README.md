# Project Images

This directory contains project screenshots and images used in the portfolio carousel.

## Current Images

- `spexop-builder.png` - Spexop Builder v0.4.3 interface screenshot
- `spexop-docs.png` - Spexop Design System documentation page
- `spexop-homepage.png` - Spexop homepage

## Image Hosting

Images in this directory are:
- Stored in `frontend/public/images/projects/`
- Copied to `dist/images/projects/` during build
- Served from GitHub Pages at: `https://olmstedian.github.io/portfolio/images/projects/`

## Adding New Images

1. Place new images in this directory
2. Use descriptive, lowercase filenames with hyphens (e.g., `project-name-feature.png`)
3. Update the database or fallback data with the image URLs
4. For GitHub Pages, use full URLs: `https://olmstedian.github.io/portfolio/images/projects/your-image.png`

## Image Optimization

Recommended image settings:
- Format: PNG or WebP
- Max width: 1920px
- Quality: 80-90%
- Compress before committing using tools like:
  - ImageOptim
  - Squoosh
  - TinyPNG
